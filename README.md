# Testing Extension Prototype

A prototype project for testing browser extensions.

## Description

This repository contains the testing extension prototype project for development and experimentation.

## Getting Started

### Prerequisites

- Node.js and npm installed
- Git for version control

### Installation

1. Clone the repository:
```bash
git clone https://github.com/joenicotra/testing-extension-prototype.git
cd testing-extension-prototype
```

2. Install dependencies (if any):
```bash
npm install
```
